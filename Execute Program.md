# Execute Program
[https://www.executeprogram.com](https://www.executeprogram.com/)

Execute Program offers interactive lessons on programming tools by [Gary Bernhardt](https://notes.andymatuschak.org/Gary_Bernhardt) (no note). The lessons interleave prose with live programming problems. Akin to the [[Mnemonic medium]], the site integrates a [[Spaced repetition memory system]] to repeat lessons’ tasks over time.

- The lesson experience: 
    - [Execute Program’s lessons unfurl their prose in response to reader interaction](https://notes.andymatuschak.org/zWeMGj1huQaESSBTmRWSvD5)
    - [Readers answer Execute Program’s prompts by executing a program](https://notes.andymatuschak.org/z9DRugkwcBv7iLRACEnWVDL)
    - [Execute Program doesn’t have non-executable prompts](https://notes.andymatuschak.org/zYCRKm7HujiD6CNtcC9gnmk) 
    - [Execute Program’s prompts act both as application prompts and recall prompts](https://notes.andymatuschak.org/zUoZHzEPeQFKaUTYctg9fqq)
    - Multiple examples can use the same code “preamble”—a useful domain-specific specialization
    - [Execute Program’s lessons are quite short](https://notes.andymatuschak.org/zLnMMGiCT6xDjMfMurTzKjs)
    - [Execute Program’s Python course has a dynamic reference](https://notes.andymatuschak.org/z6VuhFzndxKdFM7r4mV81FB)
- Course and system structure:
    - [Execute Program’s lessons don’t unlock until you’ve successfully reviewed their prerequisites](https://notes.andymatuschak.org/zHLWyC5LVXZLWE9RmoC1gua)
    - [Execute Program’s lessons assume solid recall of prior material](https://notes.andymatuschak.org/zH26uarYCm9y6tqzuxaGUzt)
    - [Missed answers in Execute Program’s reviews don’t penalize the student unless they “give up”](https://notes.andymatuschak.org/zX7RdiHGbHYnkFznUr9VZx7)
    - [Execute Program’s progress mechanics emphasize completing new lessons over increasing recall duration](https://notes.andymatuschak.org/zHJBriKPXFC2VtF8MVCdj7K)
