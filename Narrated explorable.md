# Narrated explorable
Term I’ve used to describe media akin to [[Explorable explanations]] but which integrate the narrative experience into the interactive environment, instead of separating them as is usually done.

Examples:
- [Visualizing Quaternions](https://eater.net/quaternions) by [Grant Sanderson](https://notes.andymatuschak.org/zWy4kCZkWZFxRYjW7fi6Egw) and Ben Eater 
- [Introduction to Waves](https://airladon.github.io/ivid-wave/)
- [Khan Academy’s computer programming lessons](https://www.khanacademy.org/computing/computer-programming/programming/drawing-basics/pt/making-drawings-with-code)
    - scaled up: [Scrimba](https://scrimba.com/) 
- [Cantor](https://early.khanacademy.org/cantor/), from me and my team at Khan Academy

See also [Liqvid](https://liqvidjs.org/), a framework for creating simple media of this kind.

Related: [[Dynamic mediums usually lack an authored time dimension]]
