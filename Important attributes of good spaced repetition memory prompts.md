# Important attributes of good spaced repetition memory prompts
This note collects ideas about how to encode knowledge into [[Spaced repetition memory system]] prompts, both to support memory of facts but also to foster richer understanding ([[Spaced repetition memory systems can be used to develop conceptual understanding]]).

See [How to write good prompts](https://andymatuschak.org/prompts) (2020) for a published manuscript on this topic.

Basic attributes of prompt-writing:
- [[Spaced repetition memory prompts should be concise]]. 
- [Spaced repetition memory prompts should usually focus on one atomic unit](https://notes.andymatuschak.org/zFgK9ArxAXq57iMukRsVVE9)
- [Spaced repetition memory prompts should encode ideas from multiple angles](https://notes.andymatuschak.org/zEm3fU6X2i4YViURG8cd3Zx)
    - [Spaced repetition memory prompts should connect and relate ideas](https://notes.andymatuschak.org/z3whk8UxFRFLgUQX9f7Fr2o)
- [Spaced repetition memory prompts should unambiguously produce a specific answer](https://notes.andymatuschak.org/zTYDjcneyTby8DhVGMU3zvc)
    - [Prompts must make clear what “shape” of answer they expect](https://notes.andymatuschak.org/zPKBfnnCWUEb2F8SNCWzNn8)
- [Spaced repetition memory prompts should ensure reviewers must retrieve answers from memory](https://notes.andymatuschak.org/zENFxxXLVZqyJz2qaXxuy2p)
    - [Avoid yes-no spaced repetition memory prompts](https://notes.andymatuschak.org/z4jRNWgKHL9dq1fH9qq3JhM)
- [Spaced repetition memory prompts should be written to discourage shallow “pattern matching”](https://notes.andymatuschak.org/zSiuztCZ594AZuCweRghcXT)
- [Beware set\/enumeration SRS prompts](https://notes.andymatuschak.org/Beware_set%5C%2Fenumeration_SRS_prompts)
- [Cloze deletion prompts seem to produce less understanding than question-answer pairs in spaced repetition memory systems](https://notes.andymatuschak.org/zPJt42JTcoAPTTTa2vdDonV)
- [Expert response heuristic, after Issa Rice](https://notes.andymatuschak.org/z2pzGHXwct867CYxcBwUABg)

“Covering” material:

- [Avoid orphan spaced repetition memory prompts](https://notes.andymatuschak.org/z3uFvLFRt8ognrjd6ADZMJc)

Meta / mental models:

- [A pattern language of prompt-writing](https://notes.andymatuschak.org/z8d1Rq2d13fdQCjQiWzJybD)
- [Rule of thumb: write more spaced repetition prompts than seems natural](https://notes.andymatuschak.org/zXteepWVP13Ch2snpGHHSSG)
- [Spaced repetition prompt design is about designing tasks for your future self](https://notes.andymatuschak.org/z53zJy6y76MGuJuWW4Qvab9)

Related: [The “reflected essay” metaphor for the goal for the mnemonic medium](https://notes.andymatuschak.org/zVx1Fxg5nRVskwLA3q7Wwfa)
