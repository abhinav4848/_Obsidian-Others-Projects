# **Two-component model of memory**

This model of memory suggests that the decay of a memory over time can be accurately described by two parameters: {retrievability} and {stability}.

Retrievability is defined as the {probability of a person remembering a fact after a period of time}. Stability is defined as {the durability of the memory: how long it will last if not retrieved}. Their relationship is approximated by {`R âˆ exp(-t / S)`}.
