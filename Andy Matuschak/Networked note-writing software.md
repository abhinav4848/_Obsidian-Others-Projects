# Networked note-writing software
[[Note-writing system]] emphasizing links between parts of different documents

- Block-level
    - graph-oriented (i.e. blocks are outliner nodes)
    - [Roam Research](https://notes.andymatuschak.org/zXYQBEqc4T4YypNmPWwkn1y)
    - [RemNote](https://notes.andymatuschak.org/zvZiz35n9aN8RJSEoa4yE5)
    - [Logseq](https://notes.andymatuschak.org/zL1spAm8YguH3xjcotd8CC9)
    - [Athens Research](https://notes.andymatuschak.org/zXq79C7aSnaoVv6riyVDqJM)
    - prose-oriented
    - [[Obsidian]]
        - transclusion and linking supported to individual paragraphs, heading sections
        - cannot edit transcluded content at transclusion sites
- Weak block-level support
    - [Dendron](https://notes.andymatuschak.org/zVyxT1GbaAHNcoyqHnhAoEU) (no persistently visible block reference affordances, transclusions visible only in read-only preview pane)
- Page-level
    - [NoteCards](https://notes.andymatuschak.org/zVYkBjLHq3AH6r2d5ydZktW)
    - Bear
    - [org-roam](https://www.orgroam.com/)
    - [Foam](https://foambubble.github.io/foam/)
